伪类和伪元素

在看别人写的页面的时候，在调试界面上（谷歌浏览器才有，火狐浏览器上没有）经常看见::before和::after，感到
很奇怪，css3的伪元素是单个冒号，这里怎么变成双冒号了，网上查资料说css3为了区别伪类和伪元素
将伪元素加了双冒号。现在就却别下伪元素和伪类。

伪元素

包括：

:first-letter————文本的第一个字母添加特殊样式

:first-line————文本的第一行添加特殊样式

:before————元素之前添加内容

:after————元素之后添加内容

注意：其中:before和:after两个伪元素下特有的属性 content ，用于在 CSS 渲染中向元素逻辑上的头部或尾部添加内容。注意这些添加不会改变文档内容，不会出现在 DOM 中，不可复制，仅仅是在 CSS 渲染层加入

但是它们的功用不仅于此。发现 Creative Link Effects 这个非常有意思的介绍创意链接特效的页面，里面惊人的效果大量使用到的特性除了 transform 属性进行变形之外，就是伪元素了。

既然提到 transform ，那就在看看它的用法，本人把css3那几个样式总分不清，好吧，后悔没好好学英语！现在简单区别一下吧，具体的就参考W3C吧。

transform：2D或3D转换

transition：过渡（四个参数分别是：过渡的css属性名称、过渡花费的时间，默认0、过渡效果时间曲线，默认”ease”、过渡效果何时开始）

animation：动画（@keyframes 规则）

伪类

包括：

:active————向被激活的元素添加样式

:focus————向拥有键盘输入焦点的元素添加样式

:hover————当鼠标悬浮在元素上方时，向元素添加样式

:link————向未被访问的链接添加样式

:visited————向已被访问的链接添加样式

:first-child————向元素的第一个子元素添加样式

:lang————向带有指定 lang 属性的元素添加样式

伪类可以看做是一种特殊的类选择符，是能被支持CSS的浏览器自动所识别的特殊选择符。它的最大的用处就是可以对链接在不同状态下定义不同的样式效果

伪类和类不同，是CSS已经定义好的，不能象类选择符一样随意用别的名字，根据上面的语法可以解释为对象（选择符）在某个特殊状态下（伪类）的样式

我们最常用的是4种a（锚）元素的伪类，它表示动态链接在4种不同的状态：link、visited、active、hover（未访问的链接、已访问的链接、激活链接和鼠标停留在链接上）

注意：有时这个链接访问前鼠标指向链接时有效果，而链接访问后鼠标再次指向链接时却无效果了。这是因为你把a:hover放在了a:visited的前面，这样的话由于后面的优先级高，当访问链接后就忽略了a:hover的效果。所以根据叠层顺序，我们在定义这些链接样式时，一定要按照a:link， a:visited， a:hover， a:actived的顺序书写
